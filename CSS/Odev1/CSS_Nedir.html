CSS Nedir?

“Cascading Style Sheets” yani Türkçesi “Basamaklanmış Stil Katmanları” olan CSS, HTML ile yazdığımız metin ve görüntü gibi kodların daha güzel ve renkli görünmesini sağlayan bir işaretleme dilidir. CSS, HTML ve JavaScript ile birlikte kullanıldığı zaman ortaya daha üst düzey web sayfaları ortaya çıkarabilirsiniz.

CSS Nasıl Kullanılır?

Sadece HTML ile kod yazıldığında pek bir anlam ifade etmez, işin içine CSS’ide dahil ettiğiniz zaman daha renkli ve canlı web sayfaları ortaya çıkarabilirsiniz. CSS ile yapabileceğiniz bazı şeyler;

  • Yazı rengi, büyüklüğü ve genişliği
  • Nesneleri hizalama
  • Link Özellikleri
  • Biçimlendirme
  • Liste özellikleri vb.
